## 常用方法：

#### 插入链接

```
[链接名称](链接地址)
<链接地址>
```

#### 插入图片

```
![avatar](/home/picture/1.png)
![avatar](http://baidu.com/pic/doge.png)
```
#### 插入代码
```
    ```
    <代码内容>
    ```
```
#### 插入comment

```markdown
[//]: # (我是注释，不会在浏览器中显示。)
```

#### Summary缩略效果 (Toggle list)
```
    <details>
    <summary>GET /</summary>
    </br>

    Returns a greeting message with all users and the current date/time
    ***
    </details>
```
